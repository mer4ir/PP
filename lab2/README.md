### Ключевые выводы

- **Максимальное ускорение:** 4.91x на 8 потоках (матрица 2000×2000)
- **Оптимальная конфигурация:** 4 потока для матриц < 800×800, 8 потоков для больших матриц
- **Эффективность для 2-4 потоков:** 85-98%
- **Эффективность для 8 потоков:** 43-61% (ограничено архитектурой 4+4 ядра)
