## Анализ результатов

### Ключевые наблюдения

1. **Ускорение:** 
   - Максимальное ускорение: **4.53x** (матрица 2000×2000)
   - Ускорение стабильно выше 3.38x для всех размеров ≥400

2. **Эффективность:**
   - Высокая эффективность для средних матриц (400-1200): **90-100%**
   - Для маленькой матрицы (200) эффективность ниже из-за накладных расходов MPI

3. **Производительность:**
   - Пик производительности: **1.61 GFLOPS** (матрица 400×400)
   - Снижение для больших матриц из-за кэш-промахов

### Выводы

- **MPI эффективен** для матриц размером ≥400
- **Оптимальный размер** для 4 процессов: 400×400
- **Накладные расходы** заметны только на малых размерах (200×200)
